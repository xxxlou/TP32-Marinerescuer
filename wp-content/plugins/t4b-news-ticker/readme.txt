T4B News Ticker is a flexible and easy to use WordPress plugin for news ticker. This plugin allows you to make horizontal news ticker by 4 different types of animation.


== Description ==
T4B News Ticker is a flexible and easy to use news ticker plugin for WordPress. This plugin will help you to make horizontal news ticker in your website. There are 4 different types of animation effect, like fade, slide, ticker and scroll. You can show ticker by post category or by post tags or by creating custom texts with custom links. You can write custom text in WP default post editor.

Moreover, you can control the speed of scrolling and typography. You can display ticker in your posts/pages by shortcode. You can even use the shortcode in your theme file by inserting it wherever you want to show ticker.

= Installation =

Installing  T4B News Ticker  plugin is very simple like other WordPress plugin.

* Log in as an admin on your WordPress site.
* In the menu displayed on the left, there is a "Plugins" tab. Click it.
* Now click "Add New".
* There, you have the buttons: "Upload Plugin". Click it.
* Upload the t4b-news-ticker.zip file and click Install Now button.
* After the installation, click on Activate Plugin button.
* Go to: Settings >> Ticker Settings for the options.
